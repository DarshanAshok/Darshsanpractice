Inline Block

Def of Inline-block: It's formatted just like the inline element, where it doesn't start on a new line. BUT, you can set width and height values. block The element will start on a new line and occupy the full width available. And you can set width and height values.

A block element always starts on a new line, and fills up the horizontal space left and right on the web page.

Its an span in the Inline topic.
